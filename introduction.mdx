Swiper is a free, open-source touch slider library focused on modern web and mobile applications. It delivers hardware-accelerated transitions, native-feeling swipe interactions, and a rich set of built-in modules — all with zero dependencies.

Swiper is intentionally focused on modern browsers and platforms. It is not a jQuery plugin or a legacy compatibility shim — it is purpose-built for apps that target current environments and want the best possible performance and simplicity.

## Key stats

- **~42k** GitHub stars
- **MIT** license — free for personal and commercial use
- **Zero** runtime dependencies
- **Tree-shakeable** — only the modules you use end up in your bundle
- Full **TypeScript** type definitions included

## Integration methods

Swiper supports four integration approaches. Each uses the same underlying engine and parameters.

| Method | Package import | Best for |
|---|---|---|
| **Core (Vanilla JS)** | `import Swiper from 'swiper'` | Plain HTML/JS projects, any framework |
| **React** | `import { Swiper, SwiperSlide } from 'swiper/react'` | React 16.8+ applications |
| **Vue** | `import { Swiper, SwiperSlide } from 'swiper/vue'` | Vue 3 applications |
| **Web Component** | `import 'swiper/element'` | Framework-agnostic custom elements |

<Note>
  All four integration methods share the same configuration API. Parameters documented in the API reference apply equally to Core, React, Vue, and the Web Component.
</Note>
