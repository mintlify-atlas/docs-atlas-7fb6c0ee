## Package manager

Install Swiper from npm. The package includes the core library, all modules, React and Vue components, the Web Component, TypeScript definitions, and all CSS files.

<CodeGroup>
  ```bash npm
  npm install swiper
  ```

  ```bash yarn
  yarn add swiper
  ```

  ```bash pnpm
  pnpm add swiper
  ```

  ```bash bun
  bun add swiper
  ```
</CodeGroup>

The current version is **12.1.3**. Swiper follows semantic versioning.

<Note>
  Swiper is fully tree-shakeable. Unused modules and components are excluded from your production bundle automatically when you import only what you need.
</Note>

## CDN

You can load Swiper directly from jsDelivr or unpkg without any build step. Use the bundle file, which includes all modules pre-registered.

<CodeGroup>
  ```html jsDelivr
  <!-- CSS -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@latest/swiper-bundle.min.css" />

  <!-- JS -->
  <script src="https://cdn.jsdelivr.net/npm/swiper@latest/swiper-bundle.min.js"></script>
  ```

  ```html unpkg
  <!-- CSS -->
  <link rel="stylesheet" href="https://unpkg.com/swiper@latest/swiper-bundle.min.css" />

  <!-- JS -->
  <script src="https://unpkg.com/swiper@latest/swiper-bundle.min.js"></script>
  ```
</CodeGroup>

When using the CDN bundle, `Swiper` is available as a global variable on `window`. All modules are pre-included — you do not need to pass a `modules` array.

```html
<script>
  const swiper = new Swiper('.swiper', {
    loop: true,
    pagination: { el: '.swiper-pagination' },
  });
</script>
```

<Tip>
  Pin a specific version in production to avoid unexpected breaking changes. Replace `@latest` with a version number such as `@12.1.3`.
</Tip>

## CSS imports

Swiper ships modular CSS files. Import only what you use to keep your stylesheet lean.

### Core styles

The base stylesheet is required for Swiper's layout to work. Always import this first.

```js
import 'swiper/css';
```

### Module stylesheets

Each UI module has its own stylesheet. Import the ones that match the modules you enable:

```js
import 'swiper/css';              // Required — core layout
import 'swiper/css/navigation';   // Navigation arrows
import 'swiper/css/pagination';   // Pagination dots / fraction / progress bar
import 'swiper/css/scrollbar';    // Scrollbar
import 'swiper/css/autoplay';     // Autoplay progress indicator
import 'swiper/css/effect-fade';  // Fade transition effect
import 'swiper/css/effect-cube';  // 3D Cube effect
import 'swiper/css/effect-flip';  // Flip effect
import 'swiper/css/effect-coverflow'; // Coverflow effect
import 'swiper/css/effect-cards'; // Cards effect
import 'swiper/css/effect-creative'; // Creative effect
import 'swiper/css/zoom';         // Zoom module
import 'swiper/css/thumbs';       // Thumbs module
import 'swiper/css/free-mode';    // Free mode
import 'swiper/css/grid';         // Grid layout
```

### Bundle stylesheet

If you need all styles and aren't concerned about CSS bundle size, import the full bundle:

```js
import 'swiper/css/bundle';
```

## Module imports

Swiper's modules are tree-shakeable. Import only the modules your slider actually uses.

```js
import Swiper from 'swiper';
import {
  Navigation,
  Pagination,
  Scrollbar,
  Autoplay,
  EffectFade,
  EffectCube,
  EffectFlip,
  EffectCoverflow,
  EffectCards,
  EffectCreative,
  Keyboard,
  Mousewheel,
  Virtual,
  Zoom,
  Thumbs,
  FreeMode,
  Grid,
  A11y,
  HashNavigation,
  History,
  Parallax,
  Controller,
} from 'swiper/modules';

const swiper = new Swiper('.swiper', {
  modules: [Navigation, Pagination, Autoplay],
  loop: true,
  autoplay: { delay: 3000 },
  navigation: {
    nextEl: '.swiper-button-next',
    prevEl: '.swiper-button-prev',
  },
  pagination: {
    el: '.swiper-pagination',
    clickable: true,
  },
});
```

## React and Vue imports

The React and Vue components are available as named exports from their respective entry points.

<CodeGroup>
  ```js React
  import { Swiper, SwiperSlide } from 'swiper/react';
  import { Navigation, Pagination } from 'swiper/modules';
  import 'swiper/css';
  import 'swiper/css/navigation';
  import 'swiper/css/pagination';
  ```

  ```js Vue
  import { Swiper, SwiperSlide } from 'swiper/vue';
  import { Navigation, Pagination } from 'swiper/modules';
  import 'swiper/css';
  import 'swiper/css/navigation';
  import 'swiper/css/pagination';
  ```
</CodeGroup>

## Web Component (Swiper Element)

The Swiper Web Component (custom element) can be registered globally in your entry file:

```js
import { register } from 'swiper/element/bundle';

register();
```

After registering, you can use `<swiper-container>` and `<swiper-slide>` directly in your HTML without any additional imports or initialization code. See [Usage: Element](/usage/element) for full details.

## TypeScript

TypeScript definitions are included in the `swiper` package. No separate `@types` package is needed.

```ts
import Swiper from 'swiper';
import type { SwiperOptions } from 'swiper/types';

const options: SwiperOptions = {
  slidesPerView: 3,
  spaceBetween: 24,
  loop: true,
};

const swiper = new Swiper('.swiper', options);
```
