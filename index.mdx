Swiper is a free, open-source mobile touch slider built for modern web applications. It delivers hardware-accelerated CSS transitions, a powerful module system, and first-class integrations for React, Vue, and Web Components — all with no dependencies.

## Quick start

<Steps>
  <Step title="Install Swiper">
    ```bash
    npm install swiper
    ```
  </Step>
  <Step title="Add the HTML structure">
    ```html
    <div class="swiper">
      <div class="swiper-wrapper">
        <div class="swiper-slide">Slide 1</div>
        <div class="swiper-slide">Slide 2</div>
        <div class="swiper-slide">Slide 3</div>
      </div>
    </div>
    ```
  </Step>
  <Step title="Import and initialize">
    ```js
    import Swiper from 'swiper';
    import 'swiper/css';

    const swiper = new Swiper('.swiper', {
      loop: true,
      slidesPerView: 1,
    });
    ```
  </Step>
</Steps>
